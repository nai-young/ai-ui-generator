
# 🧠 AI UI Generator

An AI-powered tool that generates modern UI components from natural language prompts using Claude API.

Built with **Next.js**, **Tailwind CSS**, and **shadcn/ui**, focused on a clean SaaS-style experience similar to tools like v0 or Linear.

---

## ✨ Features

- ⚡ Generate UI components from text prompts
- 🧩 Live preview + code view
- 🎨 Example component gallery (pricing cards, forms, dashboards)
- 🌙 Dark / light mode with custom theme
- 💾 History saved in localStorage
- ⚡ Streaming generation UX (ChatGPT-style)
- 📋 Copy code to clipboard
- 🧠 Claude-powered responses

---

## 🚀 Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Claude API (Anthropic)
